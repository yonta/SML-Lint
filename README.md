# SML Lint

Lint tool for Standard ML

## Requirement

- SML/NJ compiler (sml)
- CompilationManager (cm-build)
- ML-Yacc (ml-yacc)
- ML-Lex (ml-lex)

## Build and Usage

```shell
$ ml-build lint.cm Lint.run lint
$ ./lint [input.sml]
```
